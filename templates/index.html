About The Project

To improve decision-making transparency, this project leverages xAI to address the black‑box nature of AI. Grad‑CAM generates heatmaps to highlight key areas in medical images, enhancing comprehension. This approach increases diagnostic accuracy and helps clinicians validate model rationales.

Dataset Overview

We use Paul Mooney's Lung Disease dataset from Kaggle (View dataset). It contains 5,856 chest X-ray images across multiple categories:

Normal

1,583 images

Healthy lung reference

Viral Pneumonia

4,273 images

Cases of viral infection

This dataset provides a diverse, well-documented collection of lung X-rays to support detection, classification, and understanding of respiratory diseases. It is instrumental in advancing AI-driven healthcare and improving diagnostic outcomes.

Introduction to CNNs

Convolutional Neural Networks are designed for grid‑like data (e.g., images). They learn spatial hierarchies of features using convolutional filters, which makes them effective for classification, detection, and segmentation. Learn more here.

Input Layer

Image resized & normalized → 224×224×3

Convolution

Filters detect edges, corners, textures.

Activation

Apply ReLU → keeps positives, avoids vanishing gradient.

Pooling

Downsamples image → smaller, faster.

Fully Connected

Combines features → learns patterns.

Output

Softmax/Sigmoid → Normal vs Cancer.
